Posicionamiento

Un documento HTML consiste en un número de elementos con datos de caracteres (texto). Cuando un documento de este tipo se reproduce en una pantalla de ordenador o en una copia impresa, estos elementos generan cajas rectangulares, que definen cómo se disponen los componentes visuales que pueden ir dentro del código HTML de una página web, con el soporte de hojas de estilo CSS. Además, los navegadores aplican una versión de HTML por defecto cuando, dentro del contenido del documento, no se hace una mención explícita a la misma. Por eso, es de suma importancia que especifiquemos cuál es la versión de HTML y, como consecuencia de CSS, con la que estamos trabajando.

En el ejemplo 1 destaca el uso de elementos como colores, estilo de fuente, titulos, images, cajas, spans y parrafos. Consiste en una pagina web dedicada a mostrar algunas imagenes de la ciudad de Monterrey e historia de esas imagenes

El ejemplo 2 consiste en una pagina web que muestra imagenes de lugares que he visitado, las cajas que contienen a las imagenes estan en distintas posiciones y tamaños, aqui destaca el uso del modelo de cajas flexbox y sus estilos, margenes, padding, etc.

El ejemplo 3 consiste en una pagina web dedicada a un evento, destaca la distribucion que se hace a la caja principal la cual contiene, un navegador, una imagen central y una tabla con informacion del evento
